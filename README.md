## Cài đặt

* config database laravel `.env`
```
DB_CONNECTION=mongodb
DB_HOST=127.0.0.1
DB_PORT=27017
DB_DATABASE=web_datn
DB_USERNAME=
DB_PASSWORD=
MONGO_DB_AUTHENTICATION_DATABASE=admin
```
* install package
```
composer install
yarn install
pip -r requirements.txt
```
